# VegaProject

스카이 베가의 펌웨어 (BINX, update.zip, PDL) 다운로드 사이트입니다.
Python / Flask로 제작되었습니다.

## 구동 방법

1. 이 레포를 클론해주세요.

```
git clone https://github.com/RIPAngel/VegaProject
```

2. 필요한 것들을 설치합니다.

```
python3 -m pip install -r requirements.txt
```

3. 관리자 권한으로 실행합니다.

(리눅스의 경우)
``` 
sudo su
cd VegaProject
python3 app.py
```
